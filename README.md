This plugin allows you to hide behind blocks without any mods or other things. With it, you can play mini-games like "Hide-and-seek for blocks" or come up with your own.

To turn into a block, you need to hold down Shift and poke the block.

**Commands**
> /sk help - will help with the commands
> 
> /sk reload - reloads the config file
> 
> /sk add <block> - adds blocks that can be transformed into
> 
> /sk remove <block> - removes the block from the config
> 
> /sk list - will show all available blocks

**Permissions**
> seekblocks.use - allows you to hide behind blocks
> 
> seekblocks.admin - gives access to all plugin commands

You can also configure the plugin using the config.yml file:
> select a language
> 
> select the maximum number of disguises
> 
> delete or add blocks manually


**Java: 8 or higher**
